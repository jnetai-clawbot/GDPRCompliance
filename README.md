# GDPRCompliance — GDPR Compliance Checker

**Job 54** | **Package:** com.jnetai.gdprcompliance | **Repo:** https://github.com/jnetai-clawbot/GDPRCompliance

## What It Does
Track GDPR compliance for your business — manage processing activities, run compliance checks, assess risks, and log data breaches.

## Features
- Processing activity register (Article 30)
- Compliance checklist with pass/fail tracking
- Risk assessment scoring
- Data breach log with severity levels
- Dashboard with compliance overview
- Data export as JSON

## Build
- Android API 26+, Kotlin, Room, ViewBinding + XML, Material3 Dark
- GitHub Actions CI with release APK
